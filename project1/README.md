# Project 1: Build My World
To build and open the project: 

In folder project1:
```
$ mkdir build
$ cd build
$ cmake ../
$ make
$ export GAZEBO_PLUGIN_PATH=${GAZEBO_PLUGIN_PATH}:/home/workspace/UdacityRobotic/project1/build/
```

Back to folder project1/world
```
$ cd ../world
$ gazebo UdacityOffice
```
